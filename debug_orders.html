Debug Orders - ProfitMaker

Инструкции для отладки:

  1. Откройте браузер и перейдите к приложению ProfitMaker
  2. Откройте DevTools (F12)
  3. Перейдите на вкладку Console
  4. Добавьте виджет "User Trading Data"
  5. Перейдите на вкладку "Orders" в виджете
  6. Нажмите кнопку refresh в заголовке виджета
  7. Посмотрите на логи в консоли

Что искать в логах:

  • 🔍 [UserOrdersTab] Starting loadOrders - начало загрузки
  • 🔍 [UserOrdersTab] Accounts: - информация об аккаунтах
  • 🔍 [fetchOpenOrders] Starting fetchOpenOrders - начало CCXT запроса
  • 🔍 [fetchOpenOrders] Found account: - найденный аккаунт
  • 🔍 [fetchOpenOrders] CCXT available - CCXT доступен
  • 📊 [fetchOpenOrders] Raw orders: - полученные ордера
  • ❌ [fetchOpenOrders] Failed - ошибки

Возможные проблемы:

  • No accounts found - нет аккаунтов с API ключами
  • Account not found or missing API keys - аккаунт без ключей
  • Exchange not found in CCXT - неподдерживаемая биржа
  • Exchange does not support fetchOpenOrders - биржа не поддерживает метод
  • API Error - ошибка API биржи (неверные ключи, лимиты, etc.)
